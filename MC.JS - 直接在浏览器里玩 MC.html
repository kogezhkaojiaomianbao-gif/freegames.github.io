常见问题

1. 游戏打不开/卡死/崩溃/如何选择

请使用最新的官方 Google Chrome 浏览器访问，尽量不要使用第三方 Chrome 浏览器，不支持 Firefox 和其他浏览器内核。

如果已经是电脑 Chrome 浏览器，推荐使用 MC 1.8 WASM-GC 版，这个的性能最佳，如果是非电脑设备可以用 MC 1.8 JS 版。

2. 联机问题

MC.JS 支持且只支持局域网联机和 P2P 远程联机。

  • 局域网联机：将多个设备连接至同一个网络下，其中一个设备开启“共享世界”，其它设备的“多人游戏”里会自动搜索到开启的自定义房间。

  • P2P 远程联机：当多个设备不在同一个网络时，确保开启“共享世界”的设备有公网 IP 或 FullCone(NAT1) ，且所有设备连接至同一个中继服务器，其它设备的“多人游戏-直接连接-加入共享世界”里输入对应的邀请码就能远程联机。

MC.JS 不提供任何形式的联机服务（规避风险）。您的联机数据不经过 MC.JS 的服务器，而是 P2P 直连。若您没有公网 IP 或 FullCone(NAT1) 则无法实现远程联机（随着 IP 地址的枯竭，大部分用户都处于大内网环境，不过您可以尝试联系网络运营商开启公网），对此我们也没有任何办法。

3. 新版本（如1.13~1.21+）是否支持问题

不支持也不会支持。

这是因为技术限制，包括模组之类的也是同理无法支持。

4. 离线游戏文件下载

其它 Eaglercraft 分支项目确实有提供离线下载，但是因为 MC.JS 更新频繁，在线版本每隔一段时间会拉取最新代码进行构建，为了保证玩家都统一使用最新版本，MC.JS 暂不提供离线下载，未来可能会重新开放，敬请期待。

简介

在一台临时的电脑上想快速体验 MC ？打开 www.mc.js.cool ，不用下载 Java/启动器/客户端，不要繁琐的配置操作，打开网页就能玩！

Eaglercraft 介绍：Eaglercraft 是由 Eaglercraft 开发团队制作的在 Javascript 上运行的网页版 MC ，基于 MCJE 。使用 TeaVM 和 OpenGL 模拟器来运行与浏览器完全兼容的 Java 虚拟机。Eaglercraft 是众多网页版 MC 开源项目中完成度最高的，支持 MC 的几乎全部特性。

MC.JS 介绍：MC.JS 是专为国人优化的 Eaglercraft 的中文版，是众多 Eaglercraft 分支项目中汉化最好的，还原 MCJE 体验。MC.JS 仅为 Eaglercraft 开源项目的中文汉化分支，与 Eaglercraft 官方没有从属关系。

关于 JS 版 和 WASMGC 版：WASMGC，即 WebAssembly GC，是在最新的 Chrome 浏览器上的新技术。 采用 WASMGC 的新版本相较于 JS 版 FPS 几乎提高了 2 倍，但是如果您的设备较旧或内存不足，它可能会运行不稳定或崩溃。WASMGC 功能可能需要手动开启，有高帧率需求的可尝试 WASMGC 版。
